🏛️ Flagship Research Initiative
🇵🇰 PAKGOV-RAG (Bilingual Urdu-English Retrieval-Augmented Generation Engine)
Underlying Objective: Solving information access and linguistic gaps in administrative and legal text processing for the Pakistani government system.
Corpus Sources: Federal Board of Revenue (FBR), Higher Education Commission (HEC), NADRA registries, Securities and Exchange Commission of Pakistan (SECP), Supreme Court of Pakistan legal judgments.
Architecture Phases:
Phase 1 (Data Prep): Website scraping pipelines and multi-lingual OCR extraction (Tesseract with Urdu Nastaliq support packs).
Phase 2 (Retrieval): Hybrid semantic searching. Combined lexical BM25 and dense representation embeddings indexed using FAISS.
Phase 3 (Generation): Inference pipeline built with custom Urdu instruction-following adapter weights using QLoRA LLaMA-3/mT5.
Phase 4 (Evaluation): Metric verification with RAGAS (scoring Faithfulness, Answer Relevance, Context Precision, and Context Recall).
